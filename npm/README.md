# cch-tool

Fast CLI to navigate [Claude Code](https://claude.com/claude-code) JSONL session transcripts (`~/.claude/projects/`).

This is the npm distribution of [`cch`](https://github.com/Salourh/cch). On install it downloads the matching pre-built binary from GitHub Releases and exposes it as `cch`.

## Install

```bash
npm i -g cch-tool
cch help
cch help-all
```

## Supported platforms

- Linux x86_64, aarch64
- macOS x86_64, aarch64 (Apple Silicon)
- Windows x86_64

If your platform isn't listed, build from source: `cargo install --git https://github.com/Salourh/cch` (see the [main repo](https://github.com/Salourh/cch)).
